Объявление переменных выполняется с помощью var или val. val нельзя переназначить (он называется неизменяемым).

Пример:

fun main(args: Array<String>) {
  val num = 10;
  println(num);
  val str = "Hello";
  println(str);
}

Output:
10 Hello

Присвоение нового значения num или str приводит к ошибке компиляции.

Следующий код не будет компилироваться:

fun main(args: Array<String>) {
  val num = 10;
  num = 12; 
}

Output:
source.kt:3:3: error: val cannot be reassigned
num = 12;
^

Ключевое слово var позволяет объявлять переменные, которые можно переназначить:

fun main(args: Array<String>) {
  var test = 10;
  println(test);
  test = 21;
  println(test);
}

Output:
10 21

Как вы можете видеть, мы явно не задали тип переменных. Переменные получают соответствующий тип благодаря Kotlin's type interference, где тип автоматически выбирается из значения, присвоенного переменной.

Чтобы явно указать тип переменной, используйте следующий синтаксис:

fun main(args: Array<String>) {
  var num: Int = 8;
  val immutable_str: String = "Won't change";
  println(num);
  println(immutable_str);
}

Output:
8
Won't change

Приведенный выше код объявляет переменную num типа Int и str переменной типа String
