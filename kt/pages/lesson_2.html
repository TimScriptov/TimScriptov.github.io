Давайте начнем с создания простой программы, которая выводит на экран текст Hello, World!:

Давайте разберём код:

fun main(args: Array<String>) {
  println("Hello, World!");    
}

Output:
Hello, World!

Каждая программа Kotlin должна иметь определенную функцию main, которая является стартовой точкой для программы и выполняется с помощью ключевого слова fun.

Функция println() используется для генерации вывода. В нашем случае мы передали сообщение для вывода в двойных кавычках функции.

Теперь поработайте с кодом и попробуйте напечатать своё имя.

fun main(args: Array<String>) {
  println("Your name goes here");
}

Output:
Your name goes here

В большинстве случаев нет необходимости в точке с запятой, поскольку Kotlin предоставляет «вывод точки с запятой». Попробуйте запустить код без точки с запятой в конце инструкции println().
