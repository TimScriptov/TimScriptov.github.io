Комментирующий код - хорошая практика, помогая вам и вашим коллегам-программистам понять и улучшить код.

Kotlin есть два типа комментариев: single-line комментарии и multi-line комментарии.

Single-line комментарии начинаются с двух косых черт(//). Всё после них игнорируется компилятором:

fun main(args: Array<String") {
  println("Hello, World!"); // outputs Hello World!
}

Комментарии разъясняют намерение программы читателю.

Multi-line комментарии могут охватывать несколько строк. Они начинаются с косой черты и звездочкой, заканчиваются звездочкой и косой чертой(/* */):

/* Это многострочный комментарий */
fun main(args: Array<String>) {
  println("Hello, World!");
  /* Multi-line комментарий
  может охватывать
  несколько строк */
}
