Arrays представляет собой набор значений одного и того же типа.

Когда вам нужно сохранить список значений, например чисел, вы можете сохранить их в массиве, а не объявлять отдельные переменные для каждого числа.

Чтобы создать массив, мы можем использовать библиотечную функцию arrayOf () и передать ей значения элемента:

fun main(args: Array<String>) {
  var arr = arrayOf(1, 2, 3);
  println(arr[0]);
  println(arr[1]);
  println(arr[2]);
}

Output:
1
2
3

В массиве элементы упорядочиваются, и каждая из них имеет определенную и постоянную позицию, которая называется index.

Чтобы получить доступ к элементу массива по его индексу, следует использовать оператор index [].

fun main(args: Array<String>) {
  var arr = arrayOf(1, 2, 3);
  println(arr[0]);
  println(arr[1]);
  println(arr[2]);
}

Output:
1
2
3

Kotlin поддерживает специализированные классы для представления массивов примитивных типов. ByteArray, ShortArray, IntArray и т.д.

Ниже приведен пример объявления массива IntArray. Обратите внимание на библиотечную функцию intArrayOf (), которая создает и возвращает IntArray:

fun main(args: Array<String>) {
  var nums: IntArray = intArrayOf(4, 13, 25, 6, -2);
  println("Вывод 1-го и последнего элементов");
  println(nums[0]);
  println(nums[4]);
}

Output:
Вывод 1-го и последнего элементов
4
-2

Обратите внимание, что элементы в массиве идентифицируются с нулевого индекса, что означает то, что индекс первого элемента 0", а не один.

Таким образом, максимальный индекс 5-элементного массива равен 4.
