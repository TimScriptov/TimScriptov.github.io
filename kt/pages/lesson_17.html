В Kotlin оператор when используется для проверки нескольких условий.

Основной синтаксис оператора when выглядит следующим образом:

when (variable) {
  condition -> statement
}

when является эквивалентом оператора switch в других языках.

when соответствует его аргументу против всех ветвей последовательно до тех пор, пока не будет выполнено какое-либо условие ветвления.

fun main(args: Array<String>) {
  val num = 15;
  when (num) {
    10 -> println("Case 1");
    15 -> println("Case 2");
    20 -> println("Case 3");
    else -> println("Default case");
  }
}

when также может использоваться в качестве замены цепочки if-else if. Если аргумент не задан, условия ветвления являются просто булевыми выражениями:

fun main(args: Array<String>) {
  var age: Int = readLine()!!.toInt();
  when {
    age < 22 -> println("Too young");
    age > 50 -> println("Too old");
    else -> println("Welcome");
  }
}

Последний случай else, если все предыдущие случаи перед ним false. n n Несколько условий могут быть проверены в одном случае, если все они разделены запятыми.

fun main(args: Array<String>) {
  var x = 5;

  when(x) {
    1, 3, 5, 7, 9 -> println("Odd");
    2, 4, 6, 8 -> println("Even");
    else -> println("Неизвестно");
  }
//Вывод Odd
}

when также может использоваться как выражение. Если он используется как выражение, значение удовлетворенной ветви становится значением общего выражения.

fun main(args: Array<String>) {
  var grade = 88;
  var result = when {
    grade < 50 -> "Удовлетворительно";
    grade > 50 && grade < 80 -> "Хорошо";
    grade > 80 -> "Потрясающе!";
    else -> "Неизвестно";
  }
  println(result);
}

Если when используется как выражение, ветвь else обязательна.
