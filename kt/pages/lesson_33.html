Класс может иметь объявление функций в своем теле. Эти функции называются функциями-членов (или методы). Они объявляются так же, как и обычные функции:

class User {
  fun printHello() {
    println("Привет!");
  }
}

Функции-члены вызываются (вызываются) с точечной нотацией:

class User {
  fun printHello() {
	println("Привет!");
  }
}
fun main(args: Array<String>) {
  val obj = User();
  obj.printHello();
}

Мы можем получить доступ к свойствам класса из функций-членов, выполнить любые необходимые вычисления и т.д. Например, функция члена printHello () ниже печатает приветственное сообщение с использованием свойства name для класса:

class User {
  var name: String = "";
  fun printHello() {
    println("Привет, $name!");
  }
}
fun main(args: Array<String>) {
  val obj = User();
  obj.name = "Иван";
  obj.printHello();
}
