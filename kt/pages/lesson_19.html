Цикл do-while сначала выполняет инструкции цикла. Это означает, что инструкции в цикле do-while всегда будут выполняться хотя бы один раз, даже если условие ложное.

do {
  statements
} while (condition);

Например:

fun main(args: Array<String>) {
  var x = 5;
  do {
    println(x);
    x--;
  } while (x > 0);
}

Обратите внимание на точку с запятой в конце условия while.
