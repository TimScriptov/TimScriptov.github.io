Блок инициализатора может использоваться для автоматического запуска кода при создании объекта. Чтобы объявить блок инициализатора, используется ключевое слово init, за которым следуют фигурные скобки, содержащие код инициализации. Например:

class Student(var name: String, var age: Int) {
  init {
    println("Студент: $name");
    println("Возраст: $age");
  }
}
fun main(args: Array<String>) {
  var ivan = Student("Иван", 18);
}


Output:
Студент: Иван
Возраст: 18

Теперь, когда объект Student создан, код в блоке init будет запускаться автоматически. Класс может иметь более одного блока init. Они выполняются в том же порядке, что и в классе. Например:

class Student(name: String, age: Int){
  init {
    println("This will print first");
  }
  init {
    var sname = name;
    println("Using a variable declared above $sname");
  }
}
fun main(args: Array<String>) {
  val ivan = Student("Иван", 18)
}


Output:
This will print first
Using a variable declared above Иван

Параметры первичного конструктора могут использоваться в блоках инициализатора.
