Функция представляет собой блок кода, который может быть выполнен путем вызова функции.

Объявление функции выполняется с использованием ключевого слова fun.

Основной синтаксис функции в Kotlin выглядит следующим образом:

fun name(parameter: type = default value): return type {
  statements
}

Функция имеет имя, которое используется для ссылки на неё, другими словами, на вызов функции.

После имени появляется список функций параметров.

Следуя скобкам, это тип данных, который должен быть возвращён функцией.

Это может быть что угодно, если возвращаемая переменная соответствует типу.

Когда функция вызывается, все операторы в фигурных скобках запускаются. Давайте объявим, а затем вызовем функцию, которая печатает приветственные сообщения:

fun greetUser() {
  println("Hello user!");
  println("Good morning.");
}
	  
fun main(args: Array<String>) {
  greetUser();
}

Функция greetUser () не принимает никаких параметров и не возвращает никакого значения, поскольку она только генерирует вывод.

Определим функцию, которая принимает параметр Int и возвращает квадрат этого числа:

fun square(x: Int): Int {
  return x * x
}
	  
fun main(args: Array<String>) {
  println(square(8));
}

Ключевое слово return используется для возврата значения из функции.

Вы можете вызывать функцию столько раз, сколько захотите.

Если функция возвращает значение, вы можете назначить её переменной, например: var a = square (8);
