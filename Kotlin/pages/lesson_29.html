Kotlin предоставляет специальный тип Any.

Этот тип позволяет хранить буквально все.

Давайте объявим функцию, принимающую Any как параметр:

fun printIt(arg: Any) {
  println(arg);
}
fun main(args: Array<String>) {
  printIt("Some string");
  printIt(42);
}

Как вы можете видеть в приведенном выше примере, функцию можно вызвать с помощью любого типа аргумента.

Вы можете проверить тип аргументов с помощью оператора is.

Давайте проверим, является ли аргумент String и выводит его длину:

fun printIt(arg: Any) {
  if (arg is String) {
    println("Длина ${arg.length}");
  }
  println(arg);
}
fun main(args: Array<String>) {
  printIt("Some string");
  printIt(42);
}

Kotlin автоматически передает аргумент Any в правильный тип. В приведенном выше примере мы можем получить доступ к свойству length String без необходимости «конвертировать» arg в String.
