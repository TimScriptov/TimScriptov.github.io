Kotlin поддерживает следующие арифметические операции:

+ (сложение)

- (вычитание)

* (умножение)

/ (деление)

% (модуль)

Эти операции бинарные, что означает, что они работают с двумя операндами э

Приведенный ниже код объявляет две целые переменные и выводит их сумму, используя оператор + (сложение):

fun main(args: Array<String>) {
  var num1: Int = 8;
  var num2: Int = 3;

  // Вычисление суммы
  val sum = num1 + num2;
  println("Сумма равна $sum");
}


Output:
Сумма равна 11

Деление

Оператор деление (/) приводит к целому числу, удаляя любой остаток.

Пример:

fun main(args: Array<String>) {
  var num1: Int = 10;
  var num2: Int = 3;

  var div = num1 / num2;
  println(div);
}


Output:
3

Чтобы получить остаток от деления, используйте оператор % (модуль)

Пример:

fun main(args: Array<String>) {
  var num1: Int = 10;
  var num2: Int = 3;

  var remainder = num1 % num2;
  println(remainder);
}


Output:
1

Оператор (модуль) не может применяться к числам с плавающей запятой, однако оператор деления работает со значениями с плавающей запятой и приводит к результату с плавающей запятой:

fun main(args: Array<String>) {
  var num1: Double = 10.42;
  var num2: Double = 5.14;

  var res = num1 / num2;
  println(res);
}


Output:
2.027237354085603

Деление на ноль создает исключение.
