Строковые шаблоны используются для вставки значения переменной в строку:

fun main(args: Array<String>) {
  var x = 5;
  val s1 = "Value is $x";
  println(s1);
}


Output:
Value is 5

Все, что находится в фигурных скобках, начинающихся со знака $ (доллара), значение, и результат будет помещен в строку:

fun main(args: Array<String>) {
  var x = 5;
  var y = 6;
  val sum = "Сумма равна ${x + y}";
  println(sum);
}


Output:
Сумма равна 11

Мы можем даже вызывать функции в выражении.

Назовём функцию replace () для замены «is» на «was»:

fun main(args: Array<String>) {
  var x = 5;
  var s1 = "Value is $x";
  var s2 = "${s1.replace("is", "was")}, but now is ${x + 2}";
  println(s2);
}


Output:
Value was 5, but now is 7
