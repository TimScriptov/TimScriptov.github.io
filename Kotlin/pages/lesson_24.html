Один из самых удобных способов организовать цикл по массиву-это цикл for:

fun main(args: Array<String>) {
  var nums: IntArray = intArrayOf(4, 5, -5, 12);
  for (n in nums) {
    println("Value: $n");
  }
}

Output:
Value: 4
Value: 5
Value: -5
Value: 12

Цикл while требует дополнительной индексной переменной для доступа к элементам массива.

fun main(args: Array<String>) {
  var nums: IntArray = intArrayOf(4, 5, -5, 12);
  var index: Int = 0;
  while (index < 4) {
    println("Value: ${nums[index]}");
    ++index;
  }
}

Output:
Value: 4
Value: 5
Value: -5
Value: 12

Для итерации по массиву с индексом вы можете использовать цикл for с индексами массива

fun main(args: Array<String>) {
  var nums: IntArray = intArrayOf(4, 5, -5, 12);
  for (ix in nums.indices) {
    println("Value: ${nums[ix]}");
  }
}

Output:
Value: 4
Value: 5
Value: -5
Value: 12
