Класс похож на чертеж для объектов. Мы можем представить класс как эскиз (прототип) дома. Он содержит все детали дома (этажи, двери, окна и т.д.) На основе эскиза мы затем строим дом, который является объектом. Как дома, мы можем создать несколько объектов в классе. Для определения класса в Kotlin используется ключевое слово class:

class ClassName {

}

Теперь мы можем объявлять объекты типа ClassName ():

class ClassName {
// методы и/или свойства
}
fun main(args: Array<String>) {
  val obj = ClassName();
  var obj2: ClassName = ClassName();
}

Оба способа, показанные в приведенном выше коде, могут использоваться для создания объектов.

Каждый объект называется экземпляром класса. Процесс создания объектов называется экземпляром.
