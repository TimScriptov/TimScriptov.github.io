Может быть несколько операторов else if, когда требуется решение из трех или более действий.

Последний оператор else не является обязательным:

fun main(args: Array<String>) {
var age = readLine()!!.toInt()

  if (age < 20) {
    println("Too young")
  }
  else if (age > 50) {
    println("Too old")
  }
  else {
    println("Welcome!")
  }
}

Вы можете добавить столько операторов else if, сколько вам необходимо.
