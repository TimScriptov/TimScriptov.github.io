Параметры функции используются для передачи значений функции.

Параметры определяются в объявлении функции с использованием имени и типа параметра:

fun greetUser(name: String) {
  println("Привет $name!");
}
fun main(args: Array<String>) {
  greetUser("Max");
}

Мы объявили функцию greetUser (), которая принимает один параметр String с именем name.

Теперь при вызове функции мы можем передать действительное значение параметра функции (также называемой аргументом). В приведенном выше примере «Max» - это аргумент, переданный функции.

Функции могут принимать несколько параметров.

Давайте объявим функцию, которая принимает два целых числа и печатает их сумму:

fun printSum(num1: Int, num2: Int) {
  println("Сумма равна ${num1 + num2}!");
}
fun main(args: Array<String>) {
  val a = 4;
  val b = 5;
  printSum(a, b);
}

Одним из важнейших свойств функций является их способность возвращать значение.

Давайте объявим функцию, взяв два целочисленных параметра и вернув их.

Ключевое слово return используется для возврата значения:

fun printSum(num1: Int, num2: Int) {
println("Сумма равна ${num1 + num2}!");
}
fun getProduct(num1: Int, num2: Int): Int {
  return num1 * num2;
}
fun main(args: Array<String>) {
  val a = 4;
  val b = 5;
  printSum(a, b);
  val product = getProduct(a, b);
  println("Продукт $product");
}

Обратите внимание на дополнительный: Int после закрывающей скобки списка параметров в объявлении функции: он определяет тип возвращаемого значения функции.

Мы можем напрямую включить возвращаемое значение функции в строку, используя строковые шаблоны:

fun getProduct(num1: Int, num2: Int): Int {
  return num1 * num2;
}
fun main(args: Array<String>) {
  val a = 4;
  val b = 5;
  println("Продукт ${getProduct(a, b)}");
}
