Условные операторы выполняют разные действия в зависимости от того, какое логическое условие значение true или false.

Используйте оператор if, чтобы указать блок кода, который должен быть выполнен, если условие истинно.

Давайте выведем приветствие "Good morning!", если значение переменной hour меньше 12:

fun main(args: Array<String>) {
  val hour = 11;
  if (hour < 12) {
    println("Good morning!");
  }
}

Что делать, если нам нужно вывести другое сообщение, если hour не менее 12?

Оператор else используется для указания блока кода, который должен быть выполнен, если условие принимает значение false:

fun main(args: Array<String>) {
  val hour = 15;
  if (hour < 12) {
    println("Good morning!");
  } else {
    println("Good day!");
  }
}

Условием для оператора if может быть любое значение типа Boolean:

fun main(args: Array<String>) {
  val hour = 11;
  val isMorning: Boolean = (hour < 12);

  if (isMorning) {
    println("Доброе утро!");
  } else {
      println("Добрый день!");
  }
}

В Kotlin if - это выражение, то есть оно возвращает значение. Это означает, что мы можем использовать, если присваивать значения переменной на основе условия. Пример:

fun main(args: Array<String>) {
  val a = 8;
  val b = 42;
  val max = if (a > b) a else b;

  println(max);
}

Из-за этого в Kotlin нет тернарного оператора, поскольку такое же поведение можно выполнить с помощью оператора if.

Когда оператор if используется как выражение, предложение else является обязательным.
