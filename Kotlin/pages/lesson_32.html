Классы могут иметь свойства. Свойства похожи на обычные переменные, но объявлены в классе. Например:

class User {
  var name: String = "";
  var email: String = "";
}

Подобно регулярным переменным, свойства, объявленные с использованием ключевого слова var, являются изменчивыми, то есть они могут быть переназначены. Чтобы объявить свойство только для чтения, мы должны использовать ключевое слово val:

class User {
  var name: String = "";
  var email: String = "";
  val learningAt = "Kotlin";
}

У каждого объекта есть свои уникальные значения для его свойств класса. Чтобы использовать свойства, просто обратитесь к нему по имени объекта экземпляра, используя точку:

class User {
  var name: String = "";
  var email: String = "";
  val learningAt = "Kotlin";
}
fun main(args: Array<String>) {
  val obj = User();
  obj.name = "Иван";
  println(obj.learningAt);
}
