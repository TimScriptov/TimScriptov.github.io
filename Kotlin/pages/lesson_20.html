Оператор break завершает выполнение цикла.

fun main(args: Array<String>) {
  var x = 5;
  do {
    println(x);
    x--;
    if (x == 3) break;
  }  while (x > 0);
}

Output:
5
4

Если вы хотите оставаться в цикле, но пропустите итерацию, используйте оператор continue:

fun main(args: Array<String>) {
  var x = 0;
  while (x < 5) {
    x++;
    if (x == 3)
    continue;
    println(x);
  }
}

Output:
1
2
4
5
