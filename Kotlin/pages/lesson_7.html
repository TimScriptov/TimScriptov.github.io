Вывод в Kotlin может быть выполнен с использованием функции println.

fun main(args: Array<String>) {
  println("Hello, world!");
}

println() автоматически добавляет новую строку в конце вывода. print() может использоваться для вывода без новой строки:

fun main(args: Array<String>) {
  println("Line by");
  println("line");

  print("On the ");
  print("same line");
}

Чтобы включить значение переменной в строку, мы можем использовать либо оператор + (плюс) или строковые шаблоны, который начинается со знака доллара ($).

Ознакомьтесь с приведенными ниже примерами:

fun main(args: Array<String>) {
  var num: Int = 42;
  println("Num is" + num);
  // использование строковых шаблонов
  println("Num is $num");

  val dbl = 3.14;
  println("Int value is $num, and double value is $dbl");
}

Строковые шаблоны могут включать выражения в фигурные скобки.

Пример:

fun main(args: Array<String>) {
  var a: Int = 42;
  var b: Int = 8;

  println("Result: ${a+b}");
}"
