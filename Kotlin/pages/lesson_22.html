Kotlin предоставляет уникальный тип для ряда значений.

Диапазон имеет начало и конечное значение.

Чтобы создать диапазон, вы можете использовать функцию rangeTo ():

var myRange = 1.rangeTo(10);

Этот код определяет переменную диапазона myRange, содержащую значения от 1 до 10 включительно.

Диапазоны в Kotlin закрыты, что означает, что начальное значение и конечное значение включены в диапазон.

Чтобы создать уменьшающийся диапазон, используйте функцию downTo ():

var decRange = 8.downTo(2);

decRange содержит значения: 8, 7, 6, 5, 4, 3, 2.

Оператор ..

Другой способ определения диапазона - использовать оператор .. (double dot):

fun main(args: Array<String>) {
  var myRange = 1..10;
  println(myRange);
}

Оператор in позволяет проверить, принадлежит ли значение диапазону:

fun main(args: Array<String>) {
  var myRange = 1..10;
  if (5 in myRange) {
    println("5 находится в диапазоне");
  }
}

Аналогично, !in может использоваться для проверки того, не является ли число не принадлежащим диапазону.

Диапазоны аналогичны массивам, они представляют собой совокупности значений, которые можно было бы повторить.
