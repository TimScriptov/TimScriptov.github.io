Инкремент и декремент

Оператор инкремента ++ используется для увеличения значения переменной на единицу.

Пример:

fun main(args: Array<String>) {
  var x: Int = 4;
  x++; // тоже самое x = x + 1;
  println(x);
}


Output:
5

Аналогично, оператор декремента -- используется для уменьшения значения переменной на единицу.

fun main(args: Array<String>) {
  var x: Int = 4;
  x--; // тоже самое x = x - 1;
  println(x);
}


Output:
3

Оба оператора инкремента и декремента имеют prefix (до имени переменной) и postfix (после имени переменной).

Форма prefix увеличивает эту переменную и затем использует ее в операторе присваивания.

fun main(args: Array<String>) {
  var x: Int = 4;
  val y = ++x;
  //x увеличивается, затем присваивается y
  println(y);
}


Output:
5

Форма postfix сначала использует значение переменной, прежде чем увеличивать её.

fun main(args: Array<String>) {
  var x: Int = 4;
  val y = x++;
  // x присваивается y, затем увеличивается
  println(y);
}


Output:
4

То же самое относится к оператору декремента.
