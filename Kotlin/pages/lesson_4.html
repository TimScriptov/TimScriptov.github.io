Kotlin - это статически типизированный язык программирования с поддержкой базовых типов данных, таких как Numbers, Characters, Booleans, Arrays, and Strings.
Numbers представлены следующими встроенными типами:
- Double
- Float
- Long
- Int
- Short
- Byte
Давайте объявим Integer и выведем его значение:

fun main(args: Array<String>) {
  var num: Int = 42;
  println(num);
}

Output:
42

Типы number отличаются объемом памяти, который они выделяют:
Byte выделяет 1 байт памяти.
Short составляет 2 байта,
Int составляет 4 байта,
Long позволяет хранить 8 байтов данных.
Если вам нужно сохранить значение с плавающей точкой, например, 3.14, вы можете использовать Float или Double.


fun main(args: Array<String>) {
  var pi: Float = 3.14f;
  var e: Double = 2.71828;

  println(pi);
  println(e);
}

Output:
3.14
2.71828

Обратите внимание на f после 3.14. По умолчанию Котлин интерпретирует числа с плавающей запятой как Doubles, которые имеют более высокую точность по сравнению с Floats.
Чтобы объявить литерал Float, вы должны явно указать компилятору, что ваше значение Float, добавив f в конце значения.

Для представления логических значений (true, false) используется тип Boolean.
