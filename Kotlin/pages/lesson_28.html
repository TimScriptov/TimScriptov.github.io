Иногда удобно устанавливать значение по умолчанию для параметров функции. Эти значения по умолчанию используются, когда соответствующие аргументы опущены.

Давайте объявим функцию с тремя параметрами, которая вычисляет и возвращает их сумму:

fun getSum(num1: Int, num2: Int, num3: Int): Int {
  return num1 + num2 + num3;
}
fun main(args: Array<String>) {
  val a = 4;
  val b = 5;
  val c = 6;
  println("Сумма равна ${getSum(a, b, c)}");
}

Давайте установим значение по умолчанию для третьего аргумента:

fun getSum(num1: Int, num2: Int, num3: Int = 0): Int {
  return num1 + num2 + num3;
}
fun main(args: Array<String>) {
  val a = 4;
  val b = 5;
  val c = 6;
  println("Сумма равна ${getSum(a, b, c)}");
  println("Сумма двух чисел равна ${getSum(a, b)}");
}

Теперь мы можем вызвать одну и ту же функцию только с двумя аргументами.

Именованные аргументы

Функциональные параметры можно назвать при вызове функций:

fun getSum(num1: Int, num2: Int): Int {
  return num1 + num2;
}
fun main(args: Array<String>) {
  println(getSum(num1 = 42, num2 = 14));
}

Это делает код более читаемым при вызове функций с несколькими параметрами.

Если параметр по умолчанию предшествует параметру без значения по умолчанию, значение по умолчанию может использоваться только путем вызова функции с именованными аргументами.
