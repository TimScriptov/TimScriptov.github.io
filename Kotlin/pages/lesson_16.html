Логические операторы && (И), || (ИЛИ), ! (НЕ) используются для объединения условных операторов, образующих составное булево выражение, которое возвращает результат true или false.

Оператор &&

Оператор && (И) возвращает true, если все булевы выражения имеют значение true.

fun main(args: Array<String>) {
  var a: Int = 3;
  var b: Int = 5;

  if(a == 3 && b < 8) {
    println("hello");
  }
  if (a == 3 && b > 8) {
    println("bye");
  }
}


Output:
hello

Оператор == используется для проверки равенства.

Не равенство проверяется с помощью оператора !=.

Оператор ||

Оператор || (ИЛИ) возвращает true, если любое из булевых выражений имеет значение true.

fun main(args: Array<String>) {
  var a: Int = 3;
  var b: Int = 5;

  if(a == 3 || b < 8) {
    println("hello");
  }
  if (a == 3 || b > 8) {
    println("bye");
  }
}


Output:
hello
bye

Оператор !

Оператор ! (НЕ) возвращает обратное состояние булевого выражения.

НЕ true возвращает false, а НЕ false возвращает true.

Следующий код выводит bye, потому что (b == 6) имеет значение false, а затем изменяется на true с помощью оператора !:

fun main(args: Array<String>) {
  var b: Int = 5;

  if (!(b == 6)) {
    println("bye");
  }
}


Output:
bye
