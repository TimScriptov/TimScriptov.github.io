Простым способом чтения со стандартным вводом является функция readLine():

fun main(args: Array<String>) {
  val test = readLine();
  println("Your input is $test");
}

Все, что пользователь вводит автоматически, преобразуется в String.

Чтобы преобразовать ввод целого числа, используйте функцию toInt():

fun main(args: Array<String>) {
  var age = readLine()!!.toInt();
  println("Вам $age лет");
}

!! подразумевает, что ввод не может вернуть null, что необходимо для ввода чего-то. null означает отсутствие ценности.

Такая же логика работает для Double и других типов, используя toDouble() и аналогичные функции.

fun main(args: Array<String>) {
  var intTest: Int;
  intTest = readLine()!!.toInt();
  var dblTest = readLine()!!.toDouble();
  var longTest: Long = readLine()!!.toLong();

  println("Int input is $intTest, double input is $dblTest, long input is $longTest");
}
