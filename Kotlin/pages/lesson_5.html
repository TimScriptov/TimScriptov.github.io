Символы представлены типом Char.

fun main(args: Array<String>) {
  var ch: Char = 'a';
  println(ch);
}


Output:
a

Набор символов представлен типом String.
Давайте объявим переменную String и выведем её значение:

fun main(args: Array<String>) {
  // строковый литерал
  println("Hello, world!");

  var s: String = "Hello, world!";
  // вывод содержимого переменной s
  println(s);
}


Output:
Hello, world!
Hello, world!

В таких языках, как C ++, символы неявно рассматриваются как числа, однако в Kotlin это не так.
