Конструктор - это удобный способ передать значения свойствам объекта во время их создания. Kotlin предоставляет первичный конструктор, который можно использовать для инициализации свойств при создании объекта. Вот как определяется первичный конструктор:

class User constructor(var name: String) {
  fun sayHi() {
    println("Hello, $name");
  }
}
fun main(args: Array<String>) {
  val obj = User(readLine()!!);
  obj.sayHi();
}

Следующим именем класса является основной конструктор. Основной конструктор является частью заголовка класса. Как вы можете видеть в приведенном выше коде, конструктор определяет свойство, называемое name типа String. Теперь, создавая объект, вы можете передать значение для свойства name в фигурных скобках. Ключевое слово конструктора можно опустить:

class Student(var age: Int) {
  fun printAge() {
    println("I am $age years old");
  }
}
fun main(args: Array<String>) {
  val st = Student(21);
  st.printAge();
}

Первичные конструкторы могут объявлять свойства с помощью объявлений var или val. Множественные свойства должны быть разделены запятой. Первичные конструкторы не могут содержать код.
