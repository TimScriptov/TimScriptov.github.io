Давайте посмотрим пример массивов и циклов в действии.

Имея массив чисел с плавающей запятой, и мы хотим вычислить сумму элементов массива.

fun main(args: Array<String>) {
  var nums: DoubleArray = doubleArrayOf(3.14, 1.1, 2.2, 44.55, 21.3);
  //Вычисление суммы
  var sum: Double = 0.0;
  for (num in nums) {
    sum += num;
  }
  println(\"Сумма равна $sum\");\n
}

Мы объявили переменную сумму с начальным значением 0.0. Затем эта переменная используется для накопления суммы элементов массива с использованием цикла for.
